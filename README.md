## About EMSYX

EMSYX is a leading provider of critical software solutions for the industrial sector. We specialize in developing robust, reliable, and secure embedded systems that power mission-critical applications across various industries.

Our expertise encompasses:
- Industrial automation and control systems
- Safety-critical embedded software compliant with [ECSS standards](https://ecss.nl/)
- Real-time operating systems and middleware
- Regulatory compliance and certification support
- Cybersecurity for embedded systems following [ANSSI guidelines](https://cyber.gouv.fr/)
- Embedded systems architecture and design

## Technologies & Platforms

### Operating Systems & RTOSes
- **[FreeRTOS](https://www.freertos.org/)** - Real-time operating system for microcontrollers
- **[Zephyr](https://www.zephyrproject.org/)** - Scalable real-time operating system for IoT
- **[Yocto Project](https://www.yoctoproject.org/)** / **[Linux4Space](https://gitlab.com/linux4space)** - Custom Linux distributions for embedded systems

### Programming Languages
- **C** - Low-level system programming
- **C++** - Object-oriented embedded development
- **Rust** - Memory-safe systems programming

### Computer Vision
- **[OpenCV](https://opencv.org/)** - Open source computer vision and machine learning library

### Hardware Platforms

#### Zedboard (Xilinx Zynq-7000)
- FPGA + ARM Cortex-A9 SoC platform
- Ideal for hardware acceleration and custom IP integration
- Real-time signal processing capabilities

#### NVIDIA Jetson Nano
- GPU-accelerated edge AI computing
- CUDA support for parallel processing
- Computer vision and deep learning applications

#### STM32MP1
- Dual Cortex-A7 + Cortex-M4 heterogeneous architecture
- Linux on application processor + RTOS on Cortex-M4
- Industrial control and real-time applications

### Development & Simulation Tools
- **[QEMU](https://www.qemu.org/)** - Machine emulator and virtualizer
- **[Renode](https://renode.io/)** - Multi-node embedded system simulator
- **[Percepio Tracealyzer](https://percepio.com/)** - RTOS trace visualization and analysis

## Standards & Security

### ECSS Software Standards
We follow [European Cooperation for Space Standardization (ECSS)](https://ecss.nl/) software engineering standards for safety-critical applications:
- ECSS-E-ST-40C: Space Engineering - Software
- ECSS-Q-ST-80C: Space Product Assurance - Software product assurance

### Cybersecurity
Cybersecurity practices aligned with [ANSSI](https://cyber.gouv.fr/) (Agence Nationale de la Sécurité des Systèmes d'Information) recommendations for embedded systems security.
